groupadd — створює нову групу користувачів. 
groupmod — змінює параметри існуючої групи (назву або GID). 
useradd — створює нового користувача. 
passwd — задає або змінює пароль користувача.

who / w / last — показують інформацію про користувачів, які зараз у системі або були нещодавно.

rm / deluser / delgroup — видаляють користувача або групу.

4.1. Поняття UPG та доцільність використання 
UPG (User Private Group) — це приватна група, яка автоматично створюється для кожного нового користувача з такою ж назвою, як у користувача. 
 Використання UPG зручно, коли потрібно забезпечити ізоляцію користувачів — кожен має власну групу, і його файли не можуть змінювати інші користувачі без спеціальних прав. Це підвищує безпеку і зручність керування правами доступу. 
4.2. Команди для створення груп користувачів 
Групи створюються командою groupadd. 
 Основний синтаксис: sudo groupadd group_name 
Приклад: 
sudo groupadd super_admins 
sudo groupadd noob_users 
Можна також задати власний ідентифікатор групи (GID): sudo groupadd -g 1200 good_students 
4.3. Команди для зміни налаштувань груп 
Для зміни параметрів групи використовується команда groupmod. 
 Вона дозволяє змінити ім’я групи або її GID. 
 Приклади: 
sudo groupmod -n clerks sales      # змінює назву групи 
sudo groupmod -g 1300 super_admins # змінює GID групи 
Також переглянути інформацію про групу можна за допомогою: getent group group_name
